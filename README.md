# 🔤 CVCフラッシュカード

視覚優位のお子さん向けに作った、英語フォニックス学習用のフラッシュカードです。

**CVCは英語の基礎！** マジックeに進む前に、まずCVCで母音の概念を理解しましょう。

## 🎯 特徴

- **視覚的に分かりやすい** - 母音を色分けして、どこが変化するかひと目で分かる
- **即座にフィードバック** - 答えを選ぶとすぐに◯×が表示される

- **完全無料** - ダウンロード不要、ブラウザだけで使える

## 📖 CVCとは？

**CVC**は、英語フォニックスの基礎となる単語の型です。

### CVCの構造
- **C** = Consonant（子音）
- **V** = Vowel（母音：a, e, i, o, u）
- **C** = Consonant（子音）

**例：**
- **c-a-t** (キャット)
- **d-o-g** (ドッグ)
- **s-u-n** (サン)

### なぜCVCが大切？

✅ **母音の基本の音**（短母音）を学べる  
✅ **音を繋げて読む**練習ができる  
✅ **英語の読みの土台**になる

CVCができるようになると、もっと長い単語も読めるようになります！

## 🎵 母音の基本の音

CVCでは、母音を「短く」発音します：

| 母音 | 音 | 例 |
|------|----|----|
| **a** | ア | c**a**t, h**a**t |
| **e** | エ | b**e**d, r**e**d |
| **i** | イ | p**i**g, b**i**g |
| **o** | オ | d**o**g, h**o**t |
| **u** | ア（ウに近い） | c**u**p, s**u**n |

**注意：**アルファベット読み（エイ・イー・アイ・オウ・ユー）とは違う音です！

## 🔤 学習の前提

・アルファベットごとの「音 (sound)」の認識
　文字の「名前（例: "シー"）」ではなく、その文字が持つ「音（例: /k/）」を認識していることが最も重要です。
・音の合成 (Blending) の理解
・音の分解 (Segmenting) の理解
